# Expense_Tracker_App-
A Python-based Expense Tracker application that allows users to add, view, update, and delete daily expenses. It uses MySQL for secure data storage, supports expense categorization, and focuses on backend logic, CRUD operations, and database integration.
# Expense Tracker Desktop Application

This is a desktop-based expense tracker application.

## Features
- Add income and expenses
- View expense summary
- Store data locally

## Tech Used
- Python
- Tkinter
- SQLite

## How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
